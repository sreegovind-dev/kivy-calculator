# Kivy Calculator 🧮

A simple calculator application built using Python and Kivy with a clean graphical interface.

---

## 🚀 Features

* Basic arithmetic operations (+, -, *, /)
* Clean and responsive UI
* Input validation for operators
* Real-time calculation

---

## 🛠️ Tech Stack

* Python
* Kivy

---

## ▶️ How to Run

1. Install Kivy:

```bash
pip install kivy
```

2. Run the app:

```bash
python main.py
```
